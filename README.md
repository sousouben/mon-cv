## Lancement de l'application

```bash
#Clone the repo on your computer:
https://github.com/sousouben/mon-cv.git
```

### Installations des packages à la suite du clonage.

```bash
# avec NPM
npm install
# avec Yarn
yarn install
```

#### Lancement du projet.

```bash
# avec NPM
npm start
# avec Yarn
yarn start
```
